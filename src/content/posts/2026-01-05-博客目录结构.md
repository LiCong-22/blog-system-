# 博客系统目录结构

## 根目录结构

```
blog-system/
├── src/                 # 源代码目录
│   ├── content/         # 内容配置
│   │   └── posts/       # 博客文章 (Markdown)
│   ├── layouts/         # 页面布局
│   ├── pages/           # 页面路由
│   ├── components/      # 组件
│   └── styles/          # 样式
├── public/              # 静态资源
├── mcp-server/          # MCP 自动发布工具
├── astro.config.mjs     # Astro 配置
└── package.json
```

## 关键目录说明

### `src/content/posts/`
- 存放所有博客文章
- 使用 Markdown 格式
- 支持 frontmatter 元数据（标题、日期、标签、密码等）

### `mcp-server/`
- MCP (Model Context Protocol) 自动发布工具
- 用于快速创建和发布博客文章

## 发布流程

1. 本地开发：`npm run dev`
2. 构建生产：`npm run build`
3. 提交 GitHub → Vercel 自动部署
